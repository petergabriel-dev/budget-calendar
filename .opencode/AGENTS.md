### Dependency Injection

Use Koin for dependency injection (KMP-compatible):

```kotlin
// вњ… CORRECT - Module-based DI with Koin
val accountsModule = module {
    single<IAccountRepository> { AccountRepository(get(), get()) }
    factory { GetAccountsUseCase(get()) }
    viewModel { AccountListViewModel(get(), get()) }
}

// вќЊ INCORRECT - Direct instantiation
class AccountService {
    private val repo = AccountRepository() // Tight coupling
}
```
